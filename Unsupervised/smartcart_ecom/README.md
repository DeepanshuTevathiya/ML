This project builds a customer segmentation system for an e-commerce platform using unsupervised machine learning. The goal is to discover hidden patterns in customer behaviour and group similar customers together for better business decisions.

---

## Problem

Many e-commerce platforms use the same marketing strategy for all customers. This leads to:

- Inefficient marketing  
- Poor targeting  
- Missed opportunities to retain valuable customers  

This project solves that problem by clustering customers based on their purchasing behaviour and engagement.

---

## Dataset

The dataset contains **2240 customers with 22 features**, including:

- Demographics (age, education, income, family status)  
- Product spending  
- Purchase frequency  
- Website activity  
- Recency and complaints  

These features help identify meaningful customer segments.

---

## SmartCart Clustering System

### Approach

Project workflow:

1. Data Cleaning  
2. Feature Engineering  
3. Encoding & Scaling  
4. PCA for dimensionality reduction  
5. Finding optimal clusters (Elbow Method)  
6. K-Means Clustering  
7. Cluster analysis and insights

## Outcome

The model groups customers into different segments such as:

- High-value customers  
- Regular shoppers  
- Discount-driven buyers  
- Low-engagement users  

These insights can help businesses design targeted marketing strategies.
